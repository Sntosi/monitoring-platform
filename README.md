# 📊 Monitoring Platform | Prometheus + Grafana

A complete **real-world monitoring solution** built as a DevSecOps portfolio project.

## ✨ Features
- Real-time metrics collection with **Prometheus**
- Beautiful dashboards with **Grafana**
- Sample application exposing custom metrics
- Docker Compose full-stack setup
- Easy to extend for production use

## 🚀 Quick Start

```bash
docker compose up --build -d
```

## 🌐 Access Points:

- Application: http://localhost:8000/metrics
- Prometheus: http://localhost:9090
- Grafana: http://localhost:3000 (admin / admin123)

## 🛠️ Tech Stack

- Prometheus
- Grafana
- FastAPI (Python)
- Docker & Docker Compose

## 📈 What I Learned

- Observability best practices
- Custom metrics exposure
- Monitoring stack architecture
- Production-ready Docker setup
